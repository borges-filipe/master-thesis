\subsubsection{Baseline performance}

\textbf{Experimental Setting:} RS-Pope used as a benchmark. RS-Pope has balanced probing questions regarding objects, for example "Is there a building in the image". There are 3 difficulty levels in RS-Pope, here only the hardest one ("adversarial") was used. The questions for which the answer is "no" are treated as hallucinations. As the goal is to detect hallucinations, correctly answering "no" to a question is treated as a True Positive. The full prompt used is:

"Is there a \{object\} in the image?

A) Yes

B) No

Letter:"
